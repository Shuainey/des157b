Arrow Functions - Examples & Challenges

a named function is a function that has a specific name assigned to it. It is defined using the function keyword, followed by the chosen name for the function.

Named Function

            function addition(x,y){
                return x + y;
            }

a function expression is a way to define a function as a value and assign it to a variable or a property of an object. It involves using the function keyword without specifying a function name directly.

Function Expression

            const aVaraiable = function addition(x, y){
                return x + y;
            }

an arrow function is a concise way to write a function expression. It provides a shorter syntax compared to traditional function expressions and does not bind its own this, arguments, super, or new.target. Arrow functions are defined using the arrow (=>) syntax.

Arrow Function

            const addition = (x, y) => x + y;
